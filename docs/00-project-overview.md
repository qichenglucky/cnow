# 项目总览

## 项目名称
- 中文名：即码
- 英文名：CodeNow.ai
- 简写：Cnow

## 产品定位
即码是一个 AI 原生的集成发布平台，面向研发交付闭环，提供从服务创建、仓库初始化、CI 配置、测试环境部署、域名接入、日志接入、监控接入、生产发布、回滚治理到流程追踪的一体化能力。

## 核心原则
- 以服务为中心
- 以意图驱动配置生成
- AI 负责生成与分析，人工负责确认与决策
- 所有动作可审计、可回放、可回滚

## 已确认的架构约束
- 数据库主键统一使用自增 `bigint`
- 不预留多租户字段
- 配置草案与发布快照分离存储
- 长流程使用工作流引擎编排
- 生产动作必须走审批和审计
